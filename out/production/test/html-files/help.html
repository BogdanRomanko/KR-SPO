СПРАВКА

1. Формы программы

1.1 Главная форма программы

В главной форме программы присутствуют такие элементы как:

  • меню;
  • панель инструментов
  • контекстное меню;
  • дочерние окна;
  • строка состояния.

Главная форма программы представлена на следующем рисунке.

1.2 Все формы программы

Все формы программы при открытии их из меню "Вид" представлены на следующем рисунке.

2. Меню программы

2.1 Файл

В меню "Файл" есть несколько пунктов. Они перечислены ниже.

2.1.1 Новый файл

Пункт меню "Новый файл" создаёт новый файл для текстового редактора

2.1.2 Открыть файл

Пункт меню "Открыть файл" открывает заранее созданный файл с расширением .scpp

2.1.3 Сохранить

Пункт меню "Сохранить" сохраняет файл с расширением .scpp

2.2 Вид

В меню "Вид" есть несколько кнопок, которые выводят или убирают с главного окна определённые окна

2.5 Примеры

В меню "Примеры" существует 5 кнопок с подготовленными примерами для выполнения программы

2.6 Справка

В меню "Справка" есть две кнопки, одна из которых вызывает диалоговое окно с информацией о

программе, а вторая вызывает справочную информацию.

3. Панель инструментов программы

Панель инстурментов находится ниже главного меню программы

3.1 Новый файл

Кнопка "Новый файл" создаёт новый файл для текстового редактора

3.2 Открыть файл

Кнопка "Открыть файл" открывает заранее созданный файл с расширением .scpp

3.3 Сохранить файл

Кнопка "Сохранить" сохраняет файл с расширением .scpp

3.4 Редактирование

В правой части панели инструментов существует кнопки выделения всего текста в текстовом редакторе,

копирования выделенного текста и вставка копированного текста

4. Текстовый редактор

4.1 Вид

Вид текстового редактора вы можете увидеть ниже.

4.2 Выполнение программы

Чтобы выполнить текст программы в текстовом редакторе, необходимо нажать на кнопку "Выполнить"

4.3 Контекстное меню редактирования

В текстовром редакторе так же существует и контекстное меню, вызывать которое можно

кликом правой кнопкной мыши по текстовому редактору

5. Строка состояния

В программе предусмотрена строка состояния, в которой отображаются следующие параметры:

  • имя рабочего файла;
  • дата и время.
